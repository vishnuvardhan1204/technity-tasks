1. The two values of boolean data type are true and false.
   We write them as 'True' and 'False'.

2. The three boolean operators are and,or and not. 

3. AND Logic table:
____________________________
   Value | Value |Output
----------------------------
   True  | True  | True
   True  | False | False
   False | True  | False
   False | False | False
----------------------------

   OR Logic table:
___________________________
   Value | Value | Output
---------------------------
   True  | True  | True
   True  | False | True
   False | True  | True
   False | False | False
---------------------------
   NOT Logic table:
___________________
   Value | Output
-------------------
   True  | False
   False | True
-------------------
4. (5 > 4) and (3 == 5)                -> False
   not (5 > 4)                         -> False
   (5 > 4) or (3 == 5)                 -> True
   not ((5 > 4) or (3 == 5))           -> False  
   (True and True) and (True == False) -> False
   (not False) or (not True)           -> True
